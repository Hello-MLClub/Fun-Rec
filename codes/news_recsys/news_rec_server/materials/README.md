# 物料库

## news_scrapy/

scrapy项目结构的细节可以参考开源项目对应的文档

- 每天定时从新浪新闻上面爬取当天的新闻，并将新闻存入到mongodb数据库中
- 爬取新闻是一个增量的过程，每天只爬取当天的新闻
- 新闻爬取的时候需要提前之前物料池中的所有物料的标题给存下来用来去重

## materials_process/

- log日志处理
- 新闻画像处理
- 新闻画像存入redis
